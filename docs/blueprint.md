# **App Name**: ExpenseWise

## Core Features:

- Expense Logging: Add, edit, and delete expense records, including date, amount, category, and notes.
- Weekly Spending Chart: Visualize spending trends over the last 7 days using a bar chart.
- Monthly Spending Chart: Display spending trends for the current month using a bar or line chart.
- Category Spending Pie Chart: Show the distribution of spending across different categories using a pie chart.
- Spending Limit Gauge: Provide a visual indicator (progress bar) showing the progress towards the monthly spending limit.
- Alert Notifications: Display alert messages regarding budget limits or unusual spending patterns.
- Intelligent Spend Analyzer: Analyzes user's spending habits and gives advice on reducing expenses and staying within budget using generative AI as a tool.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability, complementing financial data presentation.
- Background color: Light blue-gray (#ECEFF1) to provide a calm and neutral backdrop.
- Accent color: Soft purple (#9575CD) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif, is used for both headings and body text, giving the application a modern and neutral aesthetic.
- Use simple, consistent icons from a library like Material Icons to represent expense categories and actions.
- Implement a responsive grid layout for the dashboard and expense list, ensuring adaptability across devices.
- Incorporate subtle transitions and animations for chart updates and user interactions.
